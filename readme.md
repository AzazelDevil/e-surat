# E-Surat

Aplikasi manajemen surat masuk & keluar berbasis web
dengan Google Spreadsheet sebagai database.

## Fitur
- Surat masuk & surat keluar
- Nomor surat input manual
- Integrasi Google Spreadsheet
- Refresh otomatis seluruh data setelah perubahan
- Bisa di-host di GitHub Pages

## Teknologi
- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Spreadsheet
